MasterPassword Release 2.1.0

Thank you for using MasterPassword for Firefox! The easiest way to handle site-unique password on the web. If you want to understand what this is all about, please see here.

You've chosen to install a beta release of this addon. Thank you for beeing an early adopter! If you find any bugs or missing features, please report them on the issue tracker.

Mozilla is fundamentally changing the environment for addons. This version adapts the webextensions api, which is the future for Firefox. For you that means that some of MasterPassword for Firefox' behaviour is changed:

Users of OS' key-vault

If you're using GNOME Keyring, KWallet, OSX Keychain or Windows Password Vault to store your master key, you'll need to download a separate helper app. Please refer to the wiki for further information.

Keyboard shortcut

It is no longer possible to change the keyboard shortcut. If this is causing you too much trouble, please add your concerns to this issue

Review

If you're happy with this addon and haven't done so already, please consider rating it.
